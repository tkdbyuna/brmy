Play Activity]

Play puzzles to earn Event Pt 'Cracker' and receive rewards. Players can earn Event Pts in the regular playlists. Players can use up to 3 stamina to clear a puzzle. The amount of stamina players use will multiply the drop rate of Pts and Medals.

Event Bonus]

  • Bonus applied to Pts gains, up to a maximum of 200%.
  • Limit Break Card doesn't accumulate bonuses.
  • All Event rewards' cards, which have been released up to date, are 25% UP!

Daily Mission]

Refresh every morning at 4 AM JST:

  1. Open the Event Top page (Reward Stamina Charge Item [1] x1)
  2. Use 10 Stamina (Reward 75pt)
  3. Clear puzzles 1 times in the event playlist (Reward Urgent request file x1 )
  4. Complete 3 daily tasks (Reward 5 Crystal)

Achievement Mission]
