Leader Skill (LEADER) is the Leader Card's effect that adds more power to the Team's Cards.

When the card is set for Support, this skill changes to SUPPORT, but the effect remains the same.

Except for R Cards, all other rarities have a leader skill, yet the effect's power, target, and element vary.

The effects can be categorized by effect target or effect element. The matrix shows the relationship between them.
