Crystal]

Only in Shop, players can buy special crystals (, ), which can be used to participate in Step-up gacha to earn a guaranteed SSR gacha ticket (See more in Gacha). Gacha requires 10 paid crystals for a gacha roll, however, players can only use them one times every day and it will daily reset at 4:00 JST.

Depending on events or special occasions, limited crystal packs will be introduced, so be sure to check the great deal here.

Buying packs from shop using the Coly ID will help players accumulate points, which can be used to earn more discounts.

The buying limit will be reset at 4:00 AM JST on the 1st day of each month.

Bremile Pass]

With 1000¥ per month, Bremile Pass will unlock the 500 crystals rewards in Bremile Mission, as well as Outfit and Mini Outfit of two different characters. Bremile Pass also:

  • Daily skip ticket increase from 1 to 3
  • Slightly boost the auto-play feature in puzzle game.
